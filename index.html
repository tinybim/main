TinyBIM 是一个基于新一代图像api WebGPU的BIM轻量化引擎

可以流畅加载超百万构件、10亿顶点场景

使用数据分块,按需加载,加载速度极快

渲染时使用动态合批,LOD剔除,视锥剔除,遮挡剔除,渲染非常流畅

定制内存压缩,显存压缩方案,资源占用更小

目前支持Revit2018-Revit2024版本,以及Ifc.更多格式正在开发中

注册即可使用，支持完全离线模式

技术支持qq群:741088338

显示模式:

支持pbr模式,着色模式,纹理模式,隐藏线模式

图纸解析:

支持Rvt内部图纸解析,并在场景中显示

多模型加载:

支持同时加载多个模型

剖切:

支持范围框剖切,和面剖切(api支持)

测量:

支持关键点捕捉,端点,中点,最近点

火焰特效:

支持火焰模拟

构建显隐、材质修改、属性查看、场景树、光照设置、

纹理显示、边线设置、管道流向...

npm install tinybim
